# Novel Skills - Claude Code 插件

中文 | [English](README.md)

为 Claude Code 打造的完整 AI 小说创作系统。从分卷规划到章节生成、质量审查和数据分析 - 一站式小说写作平台。

## 功能特性

- **项目管理**：初始化、配置和管理小说项目
- **分卷规划**：规划 100-200 章的故事结构
- **章节生成**：AI 辅助写作，质量可控
- **质量审查**：五维评分系统（人设、世界观、逻辑、文风、重复度）
- **记忆管理**：自动提取事实并更新知识库
- **角色发展**：追踪角色、关系线和成长弧
- **世界观构建**：创建和维护一致的世界体系
- **数据分析**：文本统计、情感曲线、节奏分析
- **自动化流程**：端到端的生成工作流

## 核心原则

### 故事完整性优先

本插件遵循原则：**"宁可平庸，不可崩坏"**

- **无大纲，不执行**：先规划后写作
- **系统性大于创意**：100% 遵循大纲
- **大纲即法律**：将批准的大纲视为神圣文件
- **95% 期望符合度**：满足读者期待，避免强行反转

### 质量标准

| 维度 | 权重 | 自动接受 | 人工复审 |
|-----------|--------|-------------|-------------|
| 人设一致性 | 30% | ≥9.0 | 7.0-8.9 |
| 世界观一致性 | 25% | ≥9.0 | 7.0-8.9 |
| 逻辑性 | 20% | ≥9.0 | 7.0-8.9 |
| 文风一致性 | 15% | ≥9.0 | 7.0-8.9 |
| 非重复性 | 10% | ≥9.0 | 7.0-8.9 |
| **总体** | 100% | **≥9.0** | **≥7.0** |
